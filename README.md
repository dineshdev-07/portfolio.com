# Dinesh Portfolio

Modern React + Vite developer portfolio generated from resume content only.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Resume source is copied to `public/resume.pdf`.
- All content lives in `src/data/resumeData.js`.
- UI uses React, Tailwind CSS, and Framer Motion.
